Canteen Kitchen Panel

💡 How to Test the Sync Flow (Mockup Demonstration):

  • Open both portals in separate side-by-side browser tabs.
  • On the **Kitchen Panel**, log in using username Admin and password Admin123.
  • On the **Student App**, log in with any USN (e.g. 1RV21CS001) and place an order.
  • Notice the preorder counter incrementing and playing a chime on the Kitchen Panel dashboard.
  • On the Kitchen Panel, click **"Virtual Scan"** on the order (or go to Scan tab and scan/enter code).
  • Look at the Student App: a **Payment Drawer** has automatically popped up! Choose a payment method.
  • Once paid, the Student App flashes a **Green Tick**, and the Kitchen Panel changes status to **SUCCESS** and displays the exact item list to serve.
  • Try scanning the same code again: the system flags it as **INVALID: Already Redeemed** to prevent double collection.
  • Use the floating **Time Widget** (bottom right) to simulate clock settings (e.g., test order timeouts outside break times).
